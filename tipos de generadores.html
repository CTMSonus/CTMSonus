Generador de Tonos puros


Este generador emite tonos sinusoidales puros con una frecuencia determinada.

  • (1) Boton que permite silenciar todo el generador
  • (2) Etiqueta que muestra la frecuencia actual del generador
  • (3) Etiqueta que muestra el nivel en decibeles de ambos oidos independientemente
  • (4) Slider para mover la frecuencia del sonido emitido por el generador
  • (5) Campo de Texto donde se puede ingresar la frecuencia de manera numerica y presionando boton(6) establecer la frecuencia actual del generador
  • (6) Boton para confirmar el cambio de frecuencia establecido en el campo de texto(5)
  • (7) Boton que permite silenciar lo emitido al oido izquierdo o derecho.
  • (8) Campo de texto donde se puede ingresar de manera numerica la cantidad de DBSPL que se quiere emitir, y se confirma con el boton (9)
  • (9) Boton Boton para confirmar el cambio de nivel establecido en el campo de texto(8)
  • (10) Boton para vincular los niveles de ambos oidos izquierdo y derecho, cuando son controlados con los botones (11) y (12)
  • (11) Boton para disminuir el nivel del lado correspondiente del generador.
  • (12) Boton para aumentar el nivel del lado del lado correspondiente del generador.
